**Date** 31st December 2025

## Java Basics - Summary

- **Comments:** Use `//` for single-line and `/* ... */` for multi-line comments to improve code readability.
- **Data Types:** Common types include `int`, `double`, `char`, and `String`.
- **Variables:** Store data using meaningful names; declare and assign values appropriately.
- **Displaying Output:** Use `System.out.println()` and `+` for concatenation.
- **Arithmetic Operators:** Perform calculations with `+`, `-`, `*`, `/`, `%`; declare multiple variables in one line if needed.
- **Operator Precedence:** Parentheses first, then multiplication/division/modulus, then addition/subtraction.
- **Type Conversion:** Be aware of implicit and explicit conversions; use casting for accurate results.
- **User Input:** Use the `Scanner` class to read input from the user.
- **Code Examples:** Refer to provided Java files for practical demonstrations of these concepts.

## Java Control Flow - Summary Notes

- Control flow allows selective execution of code using conditions and loops.
- **Conditional Statements:** Use `if`, `else if`, `else`, and `switch` to make decisions in code.
- **Boolean Data Type:** `boolean` can be `true` or `false` (lowercase, not in quotes).
- **Boolean Expressions:** Use comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`) to produce boolean results.
- **Logical Operators:** Combine boolean expressions with `&&` (and), `||` (or), and `!` (not).
- **Loops:** Use `while` and `for` to repeat code based on conditions.
- **Break and Continue:** `break` exits a loop early; `continue` skips to the next iteration.
- **Switch Statement:** Simplifies multi-way branching based on variable values.
- Always validate input and handle invalid cases in control flow.
- Use meaningful conditions and keep code readable for maintainability.
